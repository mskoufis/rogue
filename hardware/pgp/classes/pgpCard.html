PgpCard¶

The PgpCard class generates log entries with the path: “pyrogue.hardware.PgpCard”

PgpCard objects in C++ are referenced by the following shared pointer typedef:

typedef boost::shared_ptr<rogue::hardware::pgp::PgpCard> rogue::hardware::pgp::PgpCardPtr¶

The class description is shown below:

class PgpCard : public rogue::interfaces::stream::Master, public rogue::interfaces::stream::Slave¶
