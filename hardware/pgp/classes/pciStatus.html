PciStatus¶

PciStatus objects in C++ are referenced by the following shared pointer typedef:

typedef std::shared_ptr<rogue::hardware::pgp::PciStatus> rogue::hardware::pgp::PciStatusPtr¶

Alias for using shared pointer as PciStatusPtr.

The class description is shown below:

class PciStatus : public PciStatus¶

PCI Status Class.

This class contains the current PCI status for the PGP card. This class is a C++ wrapper around the PciStatus structure used by the lower level driver. All structure members are exposed to Python using their original names and can be read directly.

The structure which the above class wraps is shown below. All of these members are exposed in the above class.
