Info¶

Info objects in C++ are referenced by the following shared pointer typedef:

typedef std::shared_ptr<rogue::hardware::pgp::Info> rogue::hardware::pgp::InfoPtr¶

Alias for using shared pointer as InfoPtr.

The class description is shown below:

class rogue::hardware::pgp::Info : public PgpInfo¶

PGP Card Info.

This class contains the build & version information for the PGP card. This class is a C++ wrapper around the PgpInfo structure used by the lower level driver. All structure members are exposed to Python using their original names and can be read directly.
