EvrStatus¶

EvrStatus objects in C++ are referenced by the following shared pointer typedef:

typedef std::shared_ptr<rogue::hardware::pgp::EvrStatus> rogue::hardware::pgp::EvrStatusPtr¶

Alias for using shared pointer as EvrStatusPtr.

The class description is shown below:

class EvrStatus : public PgpEvrStatus¶

EVR Status Class.

This class contains the current EVR status for one of the 8 lanes on a PGP card. This class is a C++ wrapper around the PgpEvrStatus structure used by the lower level driver. All structure members are exposed to Python using their original names and can be read directly.

The structure which the above class wraps is shown below. All of these members are exposed in the above class.

struct PgpEvrStatus¶

EVR Status, one per lane.

Subclassed by rogue::hardware::pgp::EvrStatus
