Status¶

Status objects in C++ are referenced by the following shared pointer typedef:

typedef std::shared_ptr<rogue::hardware::pgp::Status> rogue::hardware::pgp::StatusPtr¶

Alias for using shared pointer as StatusPtr.

The class description is shown below:

class Status : public PgpStatus¶

PGP Status Class.

This class contains the current PGP status for one of the 8 lanes on a PGP card. This class is a C++ wrapper around the PgpStatus structure used by the lower level driver. All structure members are exposed to Python using their original names and can be read directly.

The structure which the above class wraps is shown below. All of these members are exposed in the above class.

struct PgpStatus¶
